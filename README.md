## 工作流展示

### 快捷键

- 介绍
  - 核心： 左手操作，右手辅助
  - 程序： `Capslock+_v3.2.0` `PowerToys`
  - 键位： 仿照`Vim`基于`alt`键，核心按键 `Q W E R T`  `A S D`  `1 2 3 4`  `F1 F2 F3 F4`等
  - 软件： `Chrome`、`Jetbrains`、`VScode`、`Sublime`、`Postman`、`Obsidian` 和 `Typora`等

- 操作如下：
